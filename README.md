# Swarm FM

A mobile player app built with Flutter that allows users to listen the best FM out there. Created with the goal to make a long day easier to bear when in company of our favourite twins

---
<p align="center"><img src="https://github.com/user-attachments/assets/560ad43a-6259-4674-bdd5-2dbd98af6bb7"/></p>

## Concept

- **Streamed Audio from SwarmFM**: Stay tuned to SwarmFM even when on the go. 
- **Continuous Audio Playback**: Never stop listening with background audio support.
- **Auto-Buffering**: Automatically detects stability issues and auto-starts buffering for a flwless user experience.
- **Personalized Themes**: Change your app theme to reflect your favourite twin (or femboy).
  
---

## Built With

- **Flutter** – cross-platform app development
- **Dart** – the core programming language
- **[just_audio](https://pub.dev/packages/just_audio) & [audio_service](https://pub.dev/packages/audio_service)** – robust audio playback

---

## Features

- Streams directly from [SwarmFM](https://player.sw.arm.fm/).
- Supports background playback while the app is minimized.
- Prevents audio from stopping on device lock.
- Provides a clean, modern user interface.
- Offers a lightweight and fast installation.
